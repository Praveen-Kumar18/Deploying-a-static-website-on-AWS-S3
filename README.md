# Deploying a Static Website on AWS S3

## Introduction
A static website hosted on Amazon S3 using AWS cloud services.

## Technologies Used
- AWS S3
- HTML
- CSS

## Features
- Static Website Hosting
- Public Access
- Cloud Deployment
- Scalable Storage

## Workflow
- Create S3 Bucket
- Upload HTML/CSS Files
- Enable Static Website Hosting
- Configure Public Access
- Access Website via S3 URL

## Output
Website successfully deployed and accessible through the S3 website endpoint.
